## O Ecossistema de Assistência Integrada

Nesta aula, dividimos as ferramentas em dois grandes grupos: **Plugins de IDEs Tradicionais** e **IDEs AI-First (Nativas)**.

### Plugins e Extensões (IntelliJ / VS Code)

* **JetBrains AI (Junie):** Agente focado em contexto profundo. Vai bem no Java por entender a indexação da JetBrains, facilitando refatorações estruturais.
* **Codex (OpenAI):** Integração direta para geração de código e lógica pura.
* **Cline (Open Source):** Um agente autônomo dentro do VS Code que pode ler arquivos, rodar comandos e criar projetos do zero.
* **GitHub Copilot / Claude Dev:** Os assistentes de chat e autocomplete que agora evoluíram para o "Modo Edição" (Inline Edit).

### IDEs AI-First (Nativas)

* **Cursor:** O fork do VS Code que popularizou o conceito de *Composer* (edição de múltiplos arquivos simultâneos).
* **Antigravity (Google):** Focado em integração profunda com o ecossistema Google Cloud e Gemini, com uma UX voltada para o raciocínio agentic.
* **IBM Bob:** Especializado em ambientes corporativos, governança e padrões de arquitetura robustos.
* **Windsurf:** Uma IDE que introduz o conceito de "Flow", onde a IA e o Dev alternam o controle de forma fluida.
