Essa é a categoria de ferramentas que mais cresceu nos últimos 12 meses, saindo do conceito de "No-Code" para o **"AI-Generated Code"**. Essas ferramentas não apenas criam o visual, mas escrevem o código (geralmente em React/Next.js), configuram o deploy e gerenciam o banco de dados.

### 1. v0.dev (Vercel)

É a ferramenta de referência para criação de interfaces (UI).

* **Foco:** Frontend de alta fidelidade usando **React, Tailwind CSS e Shadcn UI**.
* **Diferencial:** Como é da Vercel, o deploy é instantâneo. Você pode copiar o código gerado ou "instalar" o componente diretamente no seu projeto via CLI.
* **Uso:** Prototipagem rápida de interfaces.
* **Acesso:** [v0.dev](https://v0.dev)

### 2. Lovable (Antigo GPT Engineer)

Focada em criar **Full-Stack Apps** funcionais, não apenas o visual.

* **Foco:** Aplicações web completas (geralmente usando React + Supabase para o backend).
* **Diferencial:** Ele consegue gerenciar a lógica de negócio e persistência de dados.
* **Uso:** Criar MVPs funcionais rapidamente.
* **Acesso:** [lovable.dev](https://lovable.dev)

### 3. Replit Agent

Uma evolução da IDE online Replit, agora com um agente autônomo.

* **Foco:** Desenvolvimento Full-Stack ponta a ponta.
* **Diferencial:** O Agente do Replit configura o ambiente, instala dependências, cria o banco de dados e coloca o app no ar. Você fala "crie um clone do Uber" e ele começa a codar e configurar o servidor simultaneamente. Obs.: Não tente isso em casa.
* **Uso:** Devs que querem delegar o setup de infra.
* **Acesso:** [replit.com](https://replit.com)

### 4. Bolt.new (StackBlitz)

Uma ferramenta de desenvolvimento "in-browser" extremamente rápida.

* **Foco:** Aplicações Full-Stack modernas.
* **Diferencial:** Ele roda um ambiente de desenvolvimento completo (Node.js) dentro do seu navegador. Você pode editar o código e ver a mudança em tempo real, com suporte a agentes que corrigem erros de compilação automaticamente.
* **Uso:** Desenvolvimento ágil sem instalar nada local.
* **Acesso:** [bolt.new](https://bolt.new)

### 5. Create.xyz

Focada em facilidade de uso para criar ferramentas internas e apps funcionais.

* **Foco:** Unir design e lógica de forma visual mas baseada em código.
* **Diferencial:** Muito forte em integrações (APIs, Google Sheets, etc.). Permite criar componentes complexos apenas descrevendo-os.
* **Uso:** Criação de ferramentas internas e protótipos.
* **Acesso:** [create.xyz](https://create.xyz)

### 6. FlutterFlow (AI Gen)

Se o foco for **Mobile Nativo** (iOS/Android).

* **Foco:** Flutter.
* **Diferencial:** Recentemente integraram IA para gerar telas e lógica de ações a partir de prompts de voz ou texto, mantendo a capacidade de exportar o código fonte limpo.
* **Uso:** Aplicativos para App Store/Google Play.
* **Acesso:** [flutterflow.io](https://flutterflow.io)
